第一部分：網路基礎架構與實體層

Infrastructure & Physical Layer

1. 網路規劃與拓樸 (Topology)

網路連結模式 (拓樸)
  • 匯流排 (Bus)：共用纜線，一處斷全網癱瘓 (早期技術)。
  • 星狀 (Star)：連接中央設備 (Switch)，單線故障不影響他人 (主流)。
  • 環狀 (Ring)：單向傳遞，使用 Token 控制傳輸權 (如 Token Ring)。
  • 網狀 (Mesh)：多重路徑連結，可靠性最高但成本高。
網路架構
主從式 (Client-Server)
資源集中管理、安全性高 (如 Web, Email Server)。
對等式 (P2P)
地位平等，資源分散、管理不易 (如 BitTorrent)。
階層式設計 (Cisco)
  • 核心層 (Core)：高速骨幹。
  • 分散層 (Distribution)：路由、VLAN、ACL。
  • 存取層 (Access)：終端接入 (L2 Switch)。

2. 傳輸媒介與乙太網路標準

規格標準 速度 線材媒介 備註 (考點)
10Base5 (粗纜) 10 Mbps 粗同軸電纜 匯流排拓樸，最大 500m (已淘汰)。
10BaseT 10 Mbps Cat 3 UTP T 代表 Twisted Pair。
1000BaseT 1 Gbps Cat 5e/6 UTP 目前主流 LAN 標準。
1000BaseLX 1 Gbps 單模光纖 傳輸距離可達 5 公里。
Cat 6a 10 Gbps Cat 6a UTP 抗干擾能力優於 Cat 6。

3. 網路設備與 OSI 模型對照

設備 OSI 層級 運作原理 關鍵特性
Repeater (中繼器) L1 實體層 訊號放大再生 無流量管理，僅延伸距離。
Hub (集線器) L1 實體層 盲目廣播 單一碰撞領域 (Collision Domain)。
Switch (交換器) L2 資料鏈結層 依 MAC 轉發 獨享頻寬，每個埠為獨立碰撞領域。
Router (路由器) L3 網路層 依 IP 路由 隔離廣播領域 (Broadcast Domain)。
Gateway (閘道器) L4-L7 協定轉換 連接不同架構網路 (翻譯官)。

第二部分：協定與網路服務

Protocols, DNS, DHCP & IPv6

1. IP 定址：IPv4 vs IPv6

IPv4 結構與分類
  • Class A: 1.x.x.x - 126.x.x.x (Mask /8)
  • Class B: 128.x.x.x - 191.x.x.x (Mask /16)
  • Class C: 192.x.x.x - 223.x.x.x (Mask /24)
  • Private IP: 如 192.168.x.x, 需搭配 NAT。

CIDR: 無等級分際路由 (如 /24)，打破傳統 Class 限制，彈性配置。

IPv6 下世代協定
  • 長度: 128 位元 (16進位表示)。
  • 簡化表頭: 移除 Checksum，提升路由器效能。
  • 位址類型:
    • Global Unicast (2001::/16): 公開 IP。
    • Link-Local (FE80::/10): 僅在連結內有效。
    • Multicast (FF00::/8): 取代廣播。
  • SLAAC: 無狀態自動組態，即插即用。

2. 核心服務：DNS 與 DHCP 詳解

DNS (網域名稱系統)

階層式資料庫：Root (.) → TLD (.com) → SLD (google) → Host (www)。

記錄 (RR)功能
A主機名 → IPv4
MX郵件伺服器
CNAME別名 (Alias)
PTR反解 (IP → 名稱)
DHCP (動態主機設定)

使用 UDP (Port 67/68)。運作四步驟 (DORA)：

  1. Discover (廣播): Client 找 Server。
  2. Offer: Server 提供 IP。
  3. Request (廣播): Client 確認租用。
  4. Acknowledge: Server 正式授權租約。

* 租約過半 (50%) 時會嘗試 Renew (續約)。

3. 傳輸層協定 (TCP/UDP)

特性 TCP UDP
連線型態 連接導向 (三向交握) 非連接式 (射後不理)
可靠性 高 (重送、排序、流量控制) 低 (不保證送達)
應用場景 HTTP, FTP, Email (需資料正確) VoIP, 直播, DNS (需速度快)
常用 Port 速查

FTP(20/21), SSH(22), Telnet(23), SMTP(25), DNS(53), DHCP(67/68), HTTP(80), POP3(110), HTTPS(443)

第三部分：無線與行動通訊

Wireless, Cellular & IoT

1. Wi-Fi 技術與 AP 架設

CSMA/CA 機制

監聽 → 等待 → RTS (請求) → CTS (允許) → 資料 → ACK (確認)。解決隱藏節點問題。

標準 代數 關鍵技術
802.11n Wi-Fi 4 MIMO (多輸入多輸出)
802.11ac Wi-Fi 5 MU-MIMO (5GHz 頻段)
802.11ax Wi-Fi 6 OFDMA (多重存取，效率高)
AP 架設實務 (Infrastructure 模式)
  1. WAN 設定：接數據機，選 PPPoE (撥號) 或 DHCP。
  2. WLAN 設定：設定 SSID 與 WPA2/WPA3 密碼。
  3. 頻道選擇：2.4G 頻段避免干擾，選擇 1, 6, 11 頻道。

2. 行動通訊與 IoT

行動通訊演進
  • 2G (GSM): 數位語音、簡訊 (SMS)。
  • 3G: 上網速度提升。
  • 4G (LTE): 全 IP 化封包交換，OFDM 技術。
  • 5G: 高頻寬 (10G)、低延遲 (1ms)、海量連結。
IoT 短距通訊
  • Bluetooth: 2.4G 跳頻 (FHSS)，低功耗 (BLE)。
  • NFC: 近場通訊 (<10cm)，高安全 (行動支付)。
  • ZigBee: 低速、低功耗，網狀拓樸 (智慧家庭)。

第四部分：封包分析與診斷

Diagnostics & Analysis

常用指令與 Wireshark

  • ping [IP] 測試連線 (ICMP)。
  • tracert 追蹤路由路徑 (TTL 原理)。
  • nslookup - 8.8.8.8 指定 DNS Server 查詢域名。
  • ipconfig /all 查 MAC、DHCP、DNS 資訊。
Wireshark 過濾器：ip.addr == x.x.x.x (鎖定設備), tcp.flags.syn == 1 (看連線請求), http.request.method == "POST" (看上傳)。

第五部分：資訊安全防禦

Information Security

攻擊類型

  • DDoS：殭屍網路癱瘓服務。
  • MITM：中間人攔截未加密數據。
  • Phishing：釣魚郵件騙取帳密。
  • IP Spoofing：偽造來源 IP (如 SYN Flood)。

防護機制

  • Wi-Fi：棄用 WEP，使用 WPA2/WPA3。
  • VPN：加密通道，隱藏 IP，安全遠端存取。
  • Firewall：狀態檢測 (Stateful)，拒絕非預期連入。

第六部分：實務考題精選

Exam Q&A Collection

Q: 10 Base T 之中的 "T" 代表什麼？

Twisted Pair (雙絞線)。

Q: DHCP 的 client 要求服務時，採用何種方式？

是採用 廣播 (Broadcast) 的方式進行 (DHCP Discover)。

Q: Proxy 與 VPN 的共同功能為何？

都可以 隱藏真實 IP。

Q: 哪一個無線傳輸技術最容易受到牆壁阻隔？

紅外線 (Infrared)，因為具有強烈方向性且無法穿透障礙物。

Q: 在 TCP 協定中，封包遺失時如何重送？

由 Server 主動重送 (基於 Timeout 或重複 ACK 機制)。

Q: 要解決非固定 IP 仍能連上網站，可以採用何種服務？

DDNS (動態 DNS)。

2. IPv6 進階架構

特性 IPv4 IPv6
位址長度 32 bits (4 Bytes) 128 bits (16 Bytes)
表頭長度 20-60 Bytes (可變) 40 Bytes (固定) - 效率高
位址類型 Unicast, Broadcast, Multicast Unicast, Multicast, Anycast (無 Broadcast)
自動設定 DHCP / APIPA SLAAC (Stateless Address Auto-config)

IPv6 關鍵位址類型：

  • Link-Local (FE80::/10)：僅在區域網路有效 (類似 IPv4 169.254.x.x)。
  • Global Unicast (2000::/3)：全球公開可路由 IP。
  • Multicast (FF00::/8)：取代廣播。
  • Loopback (::1)：等同 127.0.0.1。

3. 行動通訊演進 (1G - 5G)

世代 訊號類型 關鍵技術/應用 備註
1G 類比 (Analog) AMPS (語音通話) 安全性低，易被竊聽。
2G 數位 (Digital) GSM, CDMA (語音 + 簡訊 SMS) 開始使用 SIM 卡。GPRS (2.5G) 支援上網。
3G 數位 W-CDMA, CDMA2000 (高速上網) 支援視訊通話。
4G 全 IP 分組 LTE, OFDM (行動寬頻) 理論下載可達 100Mbps - 1Gbps。
5G 全 IP 高頻寬、低延遲、廣連結 支援 IoT, 自駕車, 遠端醫療。
